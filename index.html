MERG-DEV APT Repository

The APT repository of the Model Electonic Railway Group, which is used for hosting Raspbian packages.

The current support is for Raspbian Bookworm (Debian 12) in our repository. The repositories support ARM (32-bit).

To add the repository to your installation, run the mergdev-archive.sh script, or do the following in order:

  1. You may need to install the following packages:

    sudo apt-get update↲
    sudo apt-get install ca-certificates gpg wget↲

  2. Obtain a copy of our signing key:

    wget -O - https://emthornber.github.io/rpirepo/raspbian/gpg-pubkey.asc 2>/dev/null |↲
    gpg --dearmor - |↲
    sudo tee /usr/share/keyrings/mergdev-archive-keyring.gpg >/dev/null↲

  3. Add the repository to your sources list and update.

    For For Raspbian bookworm:

    echo 'deb [signed-by=/usr/share/keyrings/mergdev-archive-keyring.gpg] https://emthornber.github.io/rpirepo/raspbian/ bookworm main' |↲
    sudo tee /etc/apt/sources.list.d/mergdev.list >/dev/null↲
    sudo apt-get update↲

Now you can install any package from our APT repository. As an example, try installing the autowap package:

sudo apt-get install autowap↲
